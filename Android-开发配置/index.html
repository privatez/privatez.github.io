Mac 配置 JAVA_HOME 环境变量

http://www.mkyong.com/java/how-to-set-java_home-environment-variable-on-mac-os-x/

首先确定是否已经安装 Jdk 。

bash_profile -> bash 使用者
zshrc -> zsh 使用者

Mac配置adb环境变量
  1. 找到android sdk路径
    一般会在~/Library/Android/sdk底下
  2. 打开命令行，敲入命令
    vi ~/.bash_profile zsh:vi ~/.zshrc
  3. 进入vi编辑模式，添加
    export PATH=${PATH}:~/Library/Android/sdk/platform-tools:~/Library/Android/sdk/tools
  4. 命令行输入，
    source ~/.bash_profile zsh:vi ~/.zshrc
