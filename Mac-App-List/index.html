Mac App List

Posted on 2016-07-04 | Visitors:

开发类：

  • Android Studio
  • Jdk
  • Git

工具类：

  • Iterm (优秀到精致的命令行工具)
  • SourceTree (千万不要轻易使用它的差异比对功能)
  • Alfred (Mac 用户不安装这个 感觉白用)
  • Lantern (一个摇摇晃晃的梯子) / Shadowsocks(拥抱开源)
  • Sublime Text (不错的文本编辑器 甚至可以变身为一个优秀的 IDE)
  • Atom
  • Homebrew (Mac 下管理安装软件工具)
  • autojump (快速跳转文件夹)
  • oh-my-zsh ()
  • f.lux (开启屏幕暖色调保护眼睛)
  • Vysor (将手机的屏幕投影到电脑上)
  • Gource (动画显示 Git 文件提交历史)
  • Hammerspoon (窗口管理工具 需要配置config)
  • Dash (查看各种文档神器 可配合 Alfred 一起使用)
  • virtualbox (开源免费的虚拟机)
  • keka (解压缩文件)
  • OhMyStar2 (git star manage)
  • Vanilla (隐藏图标)
  • 截图 (截图啊？)
  • AppCleaner
    深度卸载
  • CheatSheet
    显示快捷键提示

应用类：

  • Nylas N1 / Foxmail / Spark / YoMail (N1 三十天试用)
  • 网易云音乐 (乐库有点少，但是私人 FM 还不错) / QQ 音乐 (QQ 出品必是精品)
  • Evernote (更新后的用户方案有点坑) / 有道云笔记 (听说新版本的非常不错)
  • Chrome (如果你连这个都不知道你真的是程序狗吗？)
  • Mou / MacDown (免费的 Macdown 编辑器)

chrome 插件

  • Vimium (全键盘控制网页浏览)
  • OctoTree (Github 看代码时，左边栏会出现一个树状结构)
  • OneTab （可瞬间把所有打开的网页集合起来 减少内存消耗）
  • Momentum (新建带有漂亮背景的网页 -> 免费版的图片资源较少)
