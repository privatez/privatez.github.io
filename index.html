什么是 RxJava RxAndroid Retrofit2.0

  • RxJava 在 GitHub 主页上的自我介绍是 “a library for composing asynchronous and event-based programs using observable sequences for the Java VM”（一个在 Java VM 上使用可观测的序列来组成异步的、基于事件的程序的库）。简而言之 RxJava 就是一个基于订阅者模式的响应式编程第三方库。
  • RxAndroid 是在 RxJava 的基础上增加一些对 Android 开发的特殊支持。
  • Retrofit 底层用 OKhttp 实现的网络请求库。

如何选择开源库

项目是写来练手还是公司项目？如果纯练手个人认为可选择一些没用过的库来使用，毕竟会使用不同的库也是不错的经验吧。如果是公司项目，最好选择一些尚在维护、使用量大、功能稳定、Issues 解决速度、依赖少、包大小适中的库。

开源库推荐

  • Android-Universal-Image-Loader
    最经典的图片加载库

  • Picasso
    项目地址：https://github.com/square/picasso
    文档介绍：http://square.github.io/picasso/

  • Glide
    项目地址：https://github.com/bumptech/glide
  • Fresco
    项目地址：https://github.com/facebook/fresco
